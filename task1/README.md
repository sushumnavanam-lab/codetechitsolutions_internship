# Weather Dashboard

A Python project that fetches real-time weather data from **OpenWeatherMap API** and visualizes it using **Matplotlib**.

## Features
- Fetches temperature, humidity, pressure, wind speed, and cloudiness.
- Supports multiple cities.
- Displays data in a clean dashboard (2x3 grid of bar charts).
- Saves the dashboard as `weather_dashboard.png`.

## Requirements
- Python 3.x
- Packages:
  - `requests`
  - `matplotlib`

Install dependencies:
```bash
pip install requests matplotlib
